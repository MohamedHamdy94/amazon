# Admin

A Simple admin dashbord

## Prerequisites (Used Technologies)

* Node.js
* Angular
* mongoDB database


## Installing

1- Clone the repository

2- In the admin folder run:
```
	npm install
	npm start
```
3- In the backend folder run:
```
	npm install
	npm start
```
4- In the frontend folder run:
```
	npm install
	npm start
```
